Documentation for portfolio website – Jan 2023 release

This release is scheduled on 1 Jan 2023 IST at 12:00 AM

The Elements

Header bar

  1. The header bar sticks to the topmost edge of the webpage by default.
  2. *While the webpage is scrolled down, the header bar will move up until it is moved outside the webpage.
  3. *While the webpage is scrolled up, the header bar will again come to its initial position.
  4. The header bar will get a horizontal scrollbar when the webpage is opened on smaller screen sizes such as mobile or tablets.

Elements of the header bar

1. *Hamburger icon at the left, is used to toggle between viewing and hiding the preview box. 2. Branding with the logo of the webpage is done. 3. The channel indicator indicates the kind of set of features that are available in the webpage. Beta channel has the features that will be rolled out soon. Alpha channel has the features that are scheduled to be released after few years. This flyout also has options to switch between the channels and to view help section which contains this documentation. 4. Nav links are used to quickly go to any section within the webpage. 5. *Online status indicator at the right indicates whether the user is online or offline. An alert box will also appear if there are any internet disconnections or if slower speeds are detected. 6. *Search bar. This feature is not included in the current release. This feature is not included in the Beta channel. 7. *Settings icon. This feature is not included in the current release. Hence not found in the Beta channel as well.

The Body

1. The most prominent element is the Preview Box. It can be found at the left side on the webpage on desktop screen. At the centre filling all the body, when viewed on smaller size screens. *This preview box shows the content related to the Tile which is hovered or clicked by the curser. 2. At the right pane, a series of headings and Tiles will be shown. Each tile indicates some specific relevant topic. *When an article is hovered for a second, all other tiles and headings in the right pane will get blurred, so that only one tile and the preview pane is in focus. 3. *In order to read more about a tile, the user needs to click on the tile and then unhide the Preview Box if it is not in the view by clicking the hamburger icon in the header bar.

Requirements

1. Browsers supported: Latest chromium-based browsers such as Microsoft Edge, Google Chrome, etc., HTML based Microsoft Edge, Mozilla Firefox, and Internet Explorer 11. 2. The features marked with * are not compatible with Internet Explorer 11.
